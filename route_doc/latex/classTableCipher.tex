\doxysubsection{Подробное описание}
Шифр табличной маршрутной перестановки 

Реализует шифрование методом табличной перестановки с заданным количеством столбцов. Запись\+: по горизонтали слева направо, сверху вниз. Чтение\+: сверху вниз, справа налево. \begin{DoxyWarning}{Предупреждения}
Поддерживает только буквы и пробелы 
\end{DoxyWarning}

Алгоритм расшифрования (обратный шифрованию)\+:
\begin{DoxyEnumerate}
\item {\bfseries{Проверка входных данных}}\+: текст не должен быть пустым и содержать только буквы и пробелы
\item {\bfseries{Создание таблицы}}\+: вычисление количества строк и создание таблицы num\+Rows × num\+Columns
\item {\bfseries{Запись в таблицу}}\+: заполнение таблицы по столбцам справа налево, сверху вниз
\item {\bfseries{Чтение из таблицы}}\+: чтение данных по строкам слева направо, сверху вниз
\end{DoxyEnumerate}

Особенности алгоритма\+:
\begin{DoxyItemize}
\item Учитывается неполнота последней строки таблицы
\item Пустые ячейки заполняются пробелами и игнорируются при чтении
\end{DoxyItemize}
